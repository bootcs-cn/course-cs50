# Mario (Less)

## 背景

在任天堂的超级马里奥兄弟游戏中，马里奥需要跳过障碍物（通常是金字塔形状的砖块）。让我们用 `#` 字符来重现这个金字塔！

## 任务

实现一个程序 `mario.c`，打印一个右对齐的半金字塔。

## 示例

```
$ ./mario
Height: 1
#
```

```
$ ./mario
Height: 2
 #
##
```

```
$ ./mario
Height: 8
       #
      ##
     ###
    ####
   #####
  ######
 #######
########
```

## 规范

1. 程序必须命名为 `mario.c`
2. 提示用户输入金字塔的高度（1-8 之间的正整数）
3. 如果用户输入的不是 1-8 之间的正整数，程序应重新提示
4. 打印右对齐的半金字塔

## 提示

<details>
<summary>点击查看提示</summary>

1. 使用 `do-while` 循环确保输入有效
2. 外层 `for` 循环控制行数
3. 第一个内层循环打印空格
4. 第二个内层循环打印 `#`

```c
for (int i = 0; i < height; i++)
{
    // 打印 (height - i - 1) 个空格
    // 打印 (i + 1) 个 #
    // 打印换行
}
```

</details>
