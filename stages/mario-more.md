# Mario (More)

## 目标

实现一个程序，打印双侧对齐的金字塔。

## 背景

在《超级马里奥兄弟》游戏结尾，马里奥需要爬上双侧金字塔形状的阶梯。让我们用代码来模拟这个场景！

## 规范

- 程序文件名必须是 `mario.c`
- 提示用户输入金字塔的高度（1-8 之间的正整数）
- 如果用户输入无效（不在 1-8 范围内），重新提示输入
- 打印双侧对齐的金字塔，两个金字塔之间用两个空格分隔
- 每一行末尾不要有多余的空格

## 示例

```
$ ./mario
Height: 4
   #  #
  ##  ##
 ###  ###
####  ####
```

```
$ ./mario
Height: 1
#  #
```

```
$ ./mario
Height: -1
Height: 0
Height: 2
 #  #
##  ##
```

## 提示

1. 先实现单侧金字塔（mario-less）
2. 然后在每行后面添加两个空格和对称的右侧金字塔
3. 注意右侧金字塔不需要前导空格

## 如何测试

```bash
bootcs check cs50/mario-more
```
